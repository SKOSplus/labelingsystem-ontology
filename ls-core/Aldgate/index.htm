Introduction

The Labeling System ontology provides a vocabulary of classes and properties for the use inside the Labeling System.

The Labeling System Ontology is available at http://labeling.link/docs/ls/reference.


Labeling System Vocabulary Overview

The following table gives a none-normative overview of the Labeling System vocabulary. This document can be referenced directly, by its own URI http://labeling.i3mainz.hs-mainz.de/vocab/index.htm, or indirectly, by content negotiation from the Labeling System namespace URI http://labeling.link/docs/ls/core#. Download the RDF representation on GitHub as RDF .
